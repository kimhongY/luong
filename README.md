## ✨ មុខងារពេញលេញ

### សម្រាប់អតិថិជន (Customer)
- 🛒 **ទំព័រលក់ទំនិញ** - បង្ហាញទំនិញតាមប្រភេទ (Category)
- 🔍 **ស្វែងរក** - ស្វែងរកទំនិញ តម្រងតាមតម្លៃ និងប្រភេទ
- 🛍️ **កន្ត្រកទំនិញ** - បន្ថែម/ដកទំនិញ, ប្រើប្រាស់ Voucher
- 💳 **ទូទាត់ជាមួយ Telegram Stars** - សុវត្ថិភាព និងលឿន
- ⭐ **Rating & Reviews** - មើល និងសរសេរការវាយតម្លៃ
- 📊 **Dashboard** - មើលប្រវត្តិការបញ្ជាទិញ និងស្ថិតិ
- 🔔 **Notifications** - ទទួលបានការជូនដំណឹងពី Order

### សម្រាប់ Admin
- ⚙️ **Admin Panel** - គ្រប់គ្រងអ្វីៗទាំងអស់
- 📦 **គ្រប់គ្រងទំនិញ** - បន្ថែម, កែប្រែ, លុបទំនិញ
- 📤 **Upload រូបភាព** - ភ្ជាប់ជាមួយ Cloudinary
- 📋 **គ្រប់គ្រងការបញ្ជាទិញ** - មើល និងផ្លាស់ប្តូរស្ថានភាព
- 📊 **ស្ថិតិ** - មើលចំណូល និងទិន្នន័យ
- 🎫 **គ្រប់គ្រង Voucher** - បង្កើត និងតាមដានកូដបញ្ចុះតម្លៃ
- 📦 **គ្រប់គ្រងស្តុក** - ពិនិត្យ និងបន្ថែមស្តុក

## 🏗️ បច្ចេកវិទ្យាប្រើប្រាស់ (Tech Stack)

| ផ្នែក | បច្ចេកវិទ្យា | ការពន្យល់ |
|---------|--------------|-------------|
| **Frontend** | HTML5, CSS3, JavaScript | Telegram Web App SDK |
| **Backend API** | Python 3.9+, Flask | RESTful API |
| **Telegram Bot** | python-telegram-bot v20+ | Bot API |
| **Database** | SQLAlchemy ORM | SQLite (Dev) / PostgreSQL (Prod) |
| **Image Storage** | Cloudinary | Cloud image hosting |
| **Hosting** | GitHub Pages + Render | ឥតគិតថ្លៃ |
